## 🎯 How to Use These Prompts Effectively

### 1. **Be Specific**
- Always paste actual code, not descriptions
- Include full error messages with stack traces
- Mention your Java experience level
- Specify which exercise you're working on

### 2. **Provide Context**
- Mention you're following PHASE1_CODING_EXERCISES.md
- Reference the arXiv Paper Curator project
- Explain your learning goal for the session
- Include relevant file paths

### 3. **Ask for Examples**
- Request code examples from the actual project
- Ask for both working and broken examples
- Request multiple approaches when available
- Ask for incremental complexity

### 4. **Request Comparisons**
- Always ask for Java comparisons when learning new concepts
- Request before/after code examples
- Ask for pros/cons of different approaches
- Compare testing strategies

### 5. **Focus on Testing**
- Always relate learning back to writing tests
- Ask how to test the concepts you're learning
- Request mocking strategies for complex code
- Ask for test organization patterns
